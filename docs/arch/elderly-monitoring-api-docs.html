老年人监护系统-接口文档

通用接口

用户登录

接口描述： 老人、监护人或管理员登录，获取访问令牌。

Path： /api/auth/login/elderly 或 /api/auth/login/guardian

Method： POST

请求参数 (Body)

参数名称类型是否必须备注
phonestring是用户手机号

返回数据

{
  "token": "mock-elderly-token-1678886400000",
  "user": {
    "id": "1",
    "name": "张建国",
    "phone": "13800138000",
    "role": "elderly",
    "age": 75,
    "emergencyPhone": "13900139000",
    "avatarUrl": "http://example.com/avatar.png",
    "address": "北京市XX区XX街道XX号"
  }
}

Web管理端接口

获取被监护人列表

接口描述： 获取当前管理员/监护人账户下所有老人的概览信息。

Path： /api/guardian/elderly

Method： GET

请求参数 (Headers)

参数名称类型是否必须备注
Authorizationstring是登录后获取的Token

返回数据

[
  {
    "id": "1",
    "name": "张建国",
    "age": 75,
    "relation": "父亲",
    "avatarUrl": "http://example.com/avatar.png",
    "phone": "13800138000",
    "currentStatus": "正常行走",
    "currentLocation": "奥林匹к公园",
    "device": {
      "batteryLevel": 80,
      "isOnline": true
    }
  }
]

获取老人详情

Path： /api/guardian/elderly/:id

Method： GET

请求参数 (Path)

参数名称类型是否必须备注
idstring是老人ID

返回数据

{
  "id": "1",
  "name": "张建国",
  "age": 75,
  "avatarUrl": "http://example.com/avatar.png",
  "phone": "13800138000",
  "currentStatus": "正常行走",
  "currentLocation": "奥林匹к公园",
  "locationCoords": { "latitude": 39.9, "longitude": 116.4 },
  "device": { "deviceId": "dev001", "name": "智能手表", "batteryLevel": 85, "isOnline": true }
}

查询报警记录

Path： /api/guardian/alerts

Method： GET

请求参数 (Query)

参数名称类型是否必须备注
elderlyIdstring否筛选指定的老人ID，'all'为全部
datestring否筛选指定日期 (格式: YYYY-MM-DD)

返回数据

[
  {
    "id": "101",
    "elderlyId": "1",
    "type": "跌倒报警",
    "elderlyName": "张建国",
    "time": "2026-03-06 10:30:00",
    "description": "检测到老人跌倒",
    "location": "家中客厅",
    "level": 3,
    "status": 0
  }
]

查询历史活动

Path： /api/stats/history/:elderlyId

Method： GET

请求参数 (Path & Query)

参数名称位置类型是否必须备注
elderlyIdPathstring是老人ID, 'all'为全部
dateQuerystring否筛选指定日期 (格式: YYYY-MM-DD)

返回数据

[
  {
    "id": "hist001",
    "status": "正常行走",
    "startTime": "09:30",
    "endTime": "10:15",
    "duration": "45分钟"
  }
]

移动端接口

老人端 (ElderlyApp)

获取个人信息

Path: /api/elderly/profile | Method: GET

获取监护人列表

Path: /api/elderly/guardians | Method: GET

上报健康状态

Path: /api/elderly/status | Method: POST

Body: { "status": "正常行走", "location": "公园" }

上报位置信息

Path: /api/elderly/location | Method: POST

Body: { "latitude": 39.9, "longitude": 116.4, "address": "奥林匹к公园" }

发起紧急呼叫 (SOS)

Path: /api/elderly/sos | Method: POST

绑定/解绑监护人

Path: /api/elderly/bind 或 /api/elderly/unbind | Method: POST

Body (bind): { "phone": "13912345678" }

Body (unbind): { "guardianId": "2" }

监护人端 (GuardianApp)

监护人端App复用Web管理端的大部分接口，例如:

  • GET /api/guardian/elderly (获取被监护人列表)
  • GET /api/guardian/elderly/:id (获取老人详情)
  • GET /api/guardian/alerts (查询报警记录)
  • GET /api/stats/history/:elderlyId (查询历史活动)

此外，也包含独立的绑定/解绑接口：

绑定/解绑老人

Path: /api/guardian/bind 或 /api/guardian/unbind | Method: POST

Body (bind): { "phone": "13800138000" }

Body (unbind): { "elderlyId": "1" }
